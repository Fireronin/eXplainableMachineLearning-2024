Homework 1 Report - Model Comparison for Wine Quality Prediction

Dataset and Problem

Dataset wine_quality contains 11 continuous variables.

  • fixed_acidity
  • volatile_acidity
  • citric_acid
  • residual_sugar
  • chlorides
  • free_sulfur_dioxide
  • total_sulfur_dioxide
  • density
  • pH
  • sulphates
  • alcohol

There is huge disproportion between classes - 4715 to 183 roughly 25.77 times.

Models

Models that were tested are:

  • LogisticRegression with 10_000 maximum number of iterations
  • RandomForestClassifier with 50 estimators
  • XGBClassifier

Additionally, all model have random_state set to enhance repeatability.

Metrics

All metrics used for testing are:

  • accuracy
  • precision
  • recall
  • F1-score

Visualizations and Analysis

In `metrics.html` I added comparison of different metrics per each model. It can be seen that the metrics differs a lot across different models.
